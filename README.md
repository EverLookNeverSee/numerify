# Numerify
Multi-Threaded Numerical Methods Implemented In Fortran, Wrapped In Python.

### Dependencies:
* Python >= 3.7
* GFortran >= 5.5.0
* PyTest >= 5.3.5
* F90Wrap >= 0.2.3
